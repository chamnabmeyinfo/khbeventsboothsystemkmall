# cPanel Deployment Archive

This folder contains all documentation and guides related to cPanel deployment.

These files are archived and not needed for local development.

When you're ready to deploy to cPanel, refer to these files:
- `DEPLOY-TO-CPANEL.md` - Step-by-step deployment guide
- `CPANEL-DATABASE-CONFIG.md` - Database configuration
- `CPANEL-ENV-SETUP.md` - Environment variable setup
