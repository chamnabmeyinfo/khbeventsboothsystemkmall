# ⚡ QUICK FIX for localhost:8000

## The Problem
Your .env file has cPanel database credentials, but you're running locally.

## ✅ Solution: Update .env File

Open: C:\xampp\htdocs\KHB\khbevents\boothsystemv1\.env

### Change These Lines:

FROM:
APP_ENV=production
APP_DEBUG=false
APP_URL=https://booths.khbevents.com
DB_HOST=localhost
DB_DATABASE=khbevents_aebooths
DB_USERNAME=khbevents_admaebooths
DB_PASSWORD="ASDasd12345$$%%%"

TO:
APP_ENV=local
APP_DEBUG=true
APP_URL=http://localhost:8000
DB_HOST=127.0.0.1
DB_DATABASE=khbevents_kmall
DB_USERNAME=root
DB_PASSWORD=

### Then Run:
php artisan config:clear
php artisan cache:clear
php artisan serve

### Visit:
http://localhost:8000

---

**Note:** Update DB_DATABASE to match your actual local database name!
